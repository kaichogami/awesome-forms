# Awesome Forms

Beautiful, open-source form templates you can use anywhere. Each template is a single self-contained HTML file — no dependencies, no build step, no signup required. Every form submits to [Sutrena](https://sutrena.com) out of the box, so you can open it in a browser and it just works.

## Usage

### Open and use immediately

Every template submits to Sutrena by default. Just open the HTML file in your browser, fill it out, and your submission goes through. No backend setup needed.

### Use with your own backend

1. Pick a template from the `templates/` directory
2. Replace the `fetch('https://sutrena.com/...')` call in the `<script>` with your own endpoint
3. Deploy anywhere — static hosting, CDN, embed in your site

Each template is a single HTML file with inline CSS and minimal vanilla JavaScript. No external stylesheets, no frameworks.
