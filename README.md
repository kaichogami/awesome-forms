# Awesome Forms

Beautiful, open-source form templates you can use anywhere. Each template is a single self-contained HTML file — no dependencies, no build step, no signup required.

## Templates

| Preview | Template | Style |
|---------|----------|-------|
| [View](./templates/neo-brutalist-waitlist/) | **Neo-Brutalist Waitlist** | Bold borders, chunky type, bright yellow accent |
| [View](./templates/minimal-contact/) | **Minimal Contact** | Clean whitespace, hairline borders, system fonts |
| [View](./templates/glassmorphism-feedback/) | **Glassmorphism Feedback** | Frosted glass on gradient, CSS star rating |
| [View](./templates/retro-newsletter/) | **Retro Newsletter** | Terminal aesthetic, monospace, dashed borders |

## Usage

1. Pick a template from the `templates/` directory
2. Open the `index.html` in your browser
3. Replace the form `action="#"` with your own backend endpoint
4. Deploy anywhere — static hosting, CDN, embed in your site

Each template is a single HTML file with inline CSS. No JavaScript, no external stylesheets, no frameworks.
